# Company Mall API (키맵몰 병원 정보 및 설문지 API)
# 설치 가이드

Node js를 필수로 설치해주세요  
Typescript를 필수로 설치해주세요  

## 모듈 설치방법

npm install  
npm install -g nodemon  
npm install -g typescript  

### `npm run dev`

[http://localhost:1200](http://localhost:1200) 포트로 서버가 가동됩니다.

### `npm run build`
타입스크립트 파일을 자바스크립트로 빌드합니다.
